# Nav-Minecraft / MCNAV

> Minecraft 服主、插件开发者与资源维护者的导航站。

🌐 网址:**[mcnav.cn](https://mcnav.cn)**

开服找资源、开发查文档、维护翻 Wiki——把散落各处的 Minecraft 服务端核心、插件 Wiki、开发文档、工具和社区资源整理到一个站里,打开即用。

## ✨ 特性

- **分类聚合** —— 服务端核心、插件 Wiki、开发文档、工具、社区一站直达,不再满屏收藏夹。
- **本地搜索** —— 按标题、描述、URL、标签过滤,关键词一搜即到。
- **昼夜主题** —— 跟随系统 / 白天 / 黑夜三态切换,熬夜开服也护眼。
- **图标清晰** —— 手写图标、域名 favicon、首字母三层 fallback,告别默认占位图。
- **Apple Clean + Minecraft Accent** —— 干净排版里带一点方块世界的味道。
- **持续维护** —— 内容持续更新,也欢迎你来补充中文 Wiki / 论坛。
